# Everything Claude Code OpenClaw 적용 플랜 (상세)

## 1) 배경
- 참고 저장소: https://github.com/affaan-m/everything-claude-code
- 목표: 해당 레포의 고효율 워크플로우를 OpenClaw 환경에 맞게 이식해서 개발 효율 극대화
- 전제: **1:1 복제**가 아니라 OpenClaw 아키텍처(스킬/훅/ACP/세션 정책)에 맞춘 **선별 포팅**이 최적

---

## 2) 결론 요약
- 적용 가능성: **높음 (실행 가능)**
- 기대 이식률: **약 70~80%**
- 기대 효과(초기):
  - 개발 사이클 속도: **+25~40%**
  - 불필요한 왕복(재작업/리뷰): 감소
  - 검증 자동화 수준: 증가

핵심 포인트:
1. ECC의 강점(에이전트 분업, 스킬 묶음, 검증 루프)을 OpenClaw의 기본 기능과 잘 매핑 가능
2. 가장 주의할 부분은 Hook/Command 체계 차이
3. 작은 단위로 점진 도입해야 실제 효율이 높아짐

---

## 3) 기능 매핑 (ECC → OpenClaw)

### A. Skills / Workflow
- ECC skills → OpenClaw `skills/`로 포팅
- 우선 포팅 대상(고효율 우선):
  - `plan`
  - `tdd-workflow`
  - `verification-loop`
  - `security-review`
  - `api-design`
  - `deployment-patterns`
  - `e2e-testing`

### B. Commands
- ECC의 `/plan`, `/tdd`, `/verify` 류 명령을 OpenClaw에서 **유저 호출 스킬**(slash-style invocation)로 정리
- 운영 원칙:
  - 자주 쓰는 명령만 먼저 노출
  - 중복/저활용 명령은 늦게 도입

### C. Agents / Delegation
- ECC sub-agent 패턴 → OpenClaw `sessions_spawn` + `runtime: "acp"`(codex/claude/opencode)로 재현
- 권장 분업 구조:
  - Planner → Implementer → Reviewer
- 대형 작업은 병렬, 소형 작업은 단일 세션(토큰/컨텍스트 효율)

### D. Hooks / Automation
- ECC hooks.json을 그대로 복사하지 않고 OpenClaw 내부 hooks 체계로 재구성
- 우선 자동화 대상:
  - 세션 시작 시 컨텍스트 로딩
  - 세션 종료 시 요약/학습 포인트 저장
  - 코드 변경 후 타입/테스트/보안 점검 트리거

## 4) 4주 실행 로드맵

## Phase 1 — 진단 + PoC (3~4일)
1. 현재 기준선 측정
   - PR 리드타임
   - 테스트 실패율
   - 리뷰 왕복 횟수
   - 토큰/비용
2. 핵심 스킬 8~12개 선별 포팅
3. `/plan`, `/tdd`, `/verify` 최소 동작 확인

**완료 기준**
- 최소 3개 명령이 실제 프로젝트에서 재현 가능
- 팀/개인 루프에서 1회 이상 실사용

## Phase 2 — 자동화 레이어 (4~5일)
1. hooks 기반 세션 시작/종료 자동화
2. 검증 루프(테스트/타입체크/보안) 표준화
3. 위험 명령 승인 게이트 강화

**완료 기준**
- 코드 수정 후 검증 루프 자동 실행 비율 상승
- 수동 반복 작업(테스트/체크 명령) 감소

## Phase 3 — 멀티에이전트 최적화 (4~5일)
1. ACP 세션 기본 하네스 정리(codex 우선, 필요 시 claude/opencode)
2. Planner→Implementer→Reviewer 분업 템플릿 적용
3. 병렬 실행 가이드 확정(언제 병렬/언제 단일)

**완료 기준**
- 중형 작업 2건 이상에서 분업 패턴 성공
- 병렬 도입 후 오히려 느려지는 케이스 제거

## Phase 4 — 측정/튜닝 (1주)
1. KPI 전후 비교
2. 저효율 스킬/명령 제거(프롬프트 비대화 방지)
3. 프로젝트별 룰셋 경량화

**완료 기준**
- 최소 3개 KPI가 개선 추세 확인
- 유지/폐기 대상 명확화
